# Noter
The only place to keep your thoughts safe.

# Features
- Add, archive and delete notes.
- Star notes important to you.
- Search for notes.
- Authentication.

## Technologies / Libraries Used
- NextJS
- Material UI
- Javascript
- Firebase

## Motivation
- Getting to understand NextJS.
